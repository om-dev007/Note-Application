# 📝 Full-Stack Notes Application

A full-stack Notes application for creating and managing personal notes with timestamps and a clean UI.

Authentication and multi-user support are currently **in progress** and will be added in upcoming updates.

## Current Functionality

* Create notes
* Edit notes
* Delete notes
* View notes list
* Timestamp tracking
* Responsive UI

## Planned Features

* User authentication (JWT)
* User-specific notes
* Protected routes
* Multi-user support

## Architecture

Frontend → API → Database
(Authentication layer in progress)

## Tech Stack

**Frontend:** React, Tailwind
**Backend:** Node.js, Express, MongoDB
