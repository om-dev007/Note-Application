# 📝 Notes Application

A full-stack Notes Application built using the MERN stack where users can securely create, update, and delete personal notes.

🌐 **Live Website**  
https://note-application-dusky.vercel.app/

⚠️ **Disclaimer:**  
This project is built only for learning and educational purposes.

---

## 🚀 Features

- User Registration & Login
- Secure Authentication using JWT
- Cookie-based authentication
- Create Notes
- Edit Notes
- Delete Notes with confirmation
- Protected Routes
- Responsive UI
- Toast Notifications

---

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Router
- Axios
- React Hot Toast

### Backend
- Node.js
- Express.js

## 🔐 Authentication Flow

User Login  
↓  
JWT Token Generated  
↓  
Token Stored in HTTP-Only Cookie  
↓  
Protected API Routes  
↓  
User can access personal notes
